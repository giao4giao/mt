2. 有关使用的配置说明

|   环境变量名   |      描述      | 是否必须 |     默认值     |
|:---------:|:------------:|:----:|:-----------:|
| USERNAME  |   mt论坛的账号    |  必填  |      无      |
| PASSWORD  |   mt论坛的密码    |  必填  |      无      |
|   SEND    |    是否发送邮件    |  选填  |   无(不发送)    |
|  SENDER   |  发送邮件的邮箱地址   |  选填  |      无      |
| MAIL_PASS | 邮箱发送申请获取的授权码 |  选填  |      无      |
| RECEIVERS |   收件人的邮箱地址   |  选填  |      无      |
| MAIL_HOST |    邮件服务器     |  选填  | smtp.qq.com |
| MAIL_PORT |    邮件服务端口    |  选填  |     465     |
